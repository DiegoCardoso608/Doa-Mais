# Doa-+

Sistema voltado para facilitar doações, conectando doadores, instituições e pessoas que precisam de ajuda por meio de uma plataforma simples, organizada e acessível.

## Sobre o Projeto

O DOA+ é uma plataforma de doação inteligente que busca conectar doadores a instituições sociais ou pessoas necessitadas. O sistema foi pensado para facilitar o processo de doação de roupas, alimentos, móveis e outros itens, permitindo que recursos que seriam descartados possam chegar a quem realmente precisa.

## Problema

Muitas pessoas desejam doar roupas, comida, móveis e outros itens, mas frequentemente não sabem quem precisa, onde doar ou como encontrar instituições confiáveis. Isso faz com que muitos itens úteis sejam descartados de forma inadequada, aumentando o desperdício e reduzindo o impacto social positivo das doações.

## Solução

O sistema DOA+ propõe uma plataforma digital que conecta doadores com ONGs, instituições sociais e pessoas necessitadas.

A ideia é permitir que instituições publiquem suas necessidades, enquanto os doadores podem visualizar oportunidades de ajuda próximas de sua localização e entrar em contato diretamente para combinar a entrega.

## Funcionalidades

* Cadastro de doadores
* Cadastro de ONGs, instituições ou pessoas necessitadas
* Publicação de necessidades de doação
* Visualização de itens necessários
* Geolocalização para encontrar doações próximas
* Chat ou sistema de contato entre doador e instituição
* Controle de itens doados
* Histórico de doações do usuário
