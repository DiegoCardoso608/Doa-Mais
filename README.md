# Doa-+

Sistema voltado para facilitar doações, conectando doadores, instituições e pessoas que precisam de ajuda por meio de uma plataforma simples, organizada e acessível.

## Funcionalidades

* Cadastro de usuários
* Cadastro de instituições
* Publicação de pedidos de doação
* Sistema de busca e filtros
* Histórico de doações
* Área de contato entre doadores e instituições

## Tecnologias

* Front-end: HTML, CSS, JavaScript
* Back-end: Definir
* Banco de dados: Definir

## Objetivo

O objetivo do DoaPlus é aproximar pessoas que desejam ajudar de quem realmente precisa, tornando o processo de doação mais rápido, seguro e transparente.
